# JavaScript的初步认识  
--------
## 一、JavaScript是前台语言，而不是后台语言  
**JavaScript运行在客户的电脑里面，而不是服务器上，所以我们称为“前台语言”。**
-------------
## 二、开始写第一个JavaScript程序  
### 1、程序书写的位置  
标签对儿，里面就是书写JavaScript程序的地方。
```javascript
<script type="text/javascript">

</script>
```
text表示什么？纯文本。没错，JavaScript也是一个纯文本的语言。
### 2、alert语句  
学习第一个语句，就是alert语句。
```javascript
<script type="text/javascript">
    alert("今天天气真好");
</script>
```
alert就是英语里面的“警报”的意思。用途就是弹出“警告框”

如果我们想弹出两次警告框，那么就要写两条语句
至于为什么alert后面有一个圆括号，为什么里面又有引号，我们现在先不管。因为你知道，只要我按照这个语法书写，功能就会实现。
程序是一句一句执行，执行完上面的语句，才能之后下面的语句
```javascript
<script type="text/javascript">
alert("今天天气真好");
alert("哈哈哈哈");
</script>
```
### 3、语法规则  
JavaScript对换行、缩进、空格不敏感。
```javascript
	<script type="text/javascript">
alert("今天蓝天白云");
alert("哈哈哈，我很高兴");
	</script>
```
等价于
```javascript
<script type="text/javascript">
    alert("今天蓝天白云");
    alert("哈哈哈，我很高兴");
</script>
```
等价于：
```javascript
<script type="text/javascript">
    alert("今天蓝天白云");alert("哈哈哈，我很高兴");
</script>
```
**每一条语句末尾要加上分号，虽然分号不是必须加的，但是为了程序今后要压缩，如果不加分号，压缩之后将不能运行了。**
### 4、注释  
//表示注释：
```javascript
<script type="text/javascript">
    //alert语句表示弹出窗口
    alert("你好");
</script>
```
也可以：
```javascript
<script type="text/javascript">
    alert("你好");	//alert语句表示弹出窗口
</script>
```
注释可以很多行，但是比较麻烦：
```javascript
<script type="text/javascript">
    //alert语句表示弹出窗口
    //一定要注意用英语的符号啊！
    //千万不要再错了！
    alert("你好"); 
</script>
```
等价于：
```javascript
<script type="text/javascript">
    /*
        alert语句表示弹出窗口
        一定要注意用英语的符号啊！
        千万不要再错了！
    */
    alert("你好"); 
</script>
```
**总结一下：**

/*   
多行注释   
多行注释  
*/

## 三、认识数字和字符串 - 直接量  
“**直接量**”也称为“字面量”，就是看见什么，它就是什么。  
简单的直接量有2种：**数字、字符串**。  

数值的直接量的表达非常简单，就是写上去就行了，不需要任何的符号：  
```javascript
1alert(886);  //886是数字，所以不需要加引号。 
``` 

字符串，就是人说的话，比如单词、句子，它们不是数字。一定要加上引号。 
```javascript 
1alert(今天天气很好);  //错误的语句，因为没有加上引号
```
正确的：  
```javascript
1alert("今天天气很好");  //今天天气很好，是字符串，必须加上引号。
```

小练习，下面的语句是否正确：  
1alert(八八六);    //错误的！因为八八六是汉字，必须加上引号  
2alert(5.67);      //正确的！因为5.67是数字，不用加上引号  
3alert("100");     //正确的！虽然100是数字，但是可以是人说的啊，所以加上引号也正确。  
```javascript
console.log("今天天气真好");  
```
console表示“控制台”，log就是“输出”

## 四、变量  
### 1、整体感知  
初中的时候，学习了一个学科“代数”，x、y、z、a、b、c。  
计算机的程序中，也是有这样的量，就是用字母来表示数字、字符串等其他东西的，称为“变量”。  
```javascript
var a = 100;
console.log(a);
```
var a = 100;  
这就是一个固定的写法，就是语法。也就是说，我们使用var来定义一个变量。  
var就是英语“variant”变量的缩写。后面要加一个空格，空格后面的东西就是“变量名”，我们可以给变量任意的取名字。  
var就是一个关键字，所谓关键字，就是有特殊功能的小词语。关键字后面一定要有空格隔开。  
等号表示赋值，会将等号右边的值，赋给左边的变量。  
### 2、变量的命名和规范  
**只能由英语字母、数字、下划线、美元符号$构成，且不能以数字开头，并且不能是JavaScript保留字。**   

下列的单词，叫做保留字，就是说不允许当做变量名，不用记：  
abstract、boolean、byte、char、class、const、debugger、double、enum、export、extends、final、float、goto
implements、import、int、interface、long、native、package、private、protected、public、short、static、super、synchronized、throws、transient、volatile   
大写字母是可以使用的，并且大小写敏感。也就是说A和a是两个变量。  
### 3、变量的定义和赋值  
变量用var来定义。只有定义之后，这个变量才能够使用。  
比如，我们不设置变量，直接输出：  **错误**
```javascript
<script type="text/javascript">
    console.log(a);
</script>
```
正确：
```javascript
var a;   // 定义
a = 100;  //赋值
console.log(a);  //输出100
```
有经验的程序员，会把定义和赋值写在一起：
```javascript
var a = 100;    //定义，并且赋值100
console.log(a);  //输出100
```
### 4、区分变量和字符串  
```javascript
var a = 100;
console.log("哥们，字母表里面第一个字母是啥？");
console.log("a");    //输出字母a
``` 
这个a在引号里面，所以就是一个字符串“a”了，而不是一个变量。换句话说，一个变量如果想输出保存的内容，那么就不能加引号。

## 五、变量的类型  
变量里面能够存储数字、字符串。变量会自动的根据存储的东西类型不同，来决定自己的类型。  
也就是说变量有类型。  
### 1、数值型  
如果一个变量中，存放了数字，那么这个变量就是数值型的。
```javascript
var a = 100;			//定义了一个变量a，并且赋值100
console.log(typeof a);	//输出a变量的类型  
```
typeof表示“某某的类型”   

**语法**：  
typeof 变量  

JavaScript种，只要是个数，那么就是数值型的，**无论整浮、无论大小、无论正负，都是number类型的**。  
“浮”就是浮点数，就是“小数”，术语叫做“浮点数”。  
```javascript
var a = 100;			//定义了一个变量a，并且赋值100
var b = 234234234324324324;  

console.log(typeof a);
console.log(typeof b);
```
**number**
### 2、字符串型  
string就是字符串型的意思。
```javascript
var a = "abcde";
var b = "传智播客";
var c = "123123";
var d = "哈哈哈哈哈";
var e = "";     //空字符串

console.log(typeof a);
console.log(typeof b);
console.log(typeof c);
console.log(typeof d);
console.log(typeof e);
```
**spring**
### 3、连字符和加号  
键盘上+在JS中有两层含义：
1） 连字符
2） 加
```javascript
console.log("我" + "爱" + "你");	//连字符，把三个独立的汉字，连接在一起了
console.log("我+爱+你");			//原样输出
console.log(1+2+3);				  //输出6  
```
同样是加号，有时候表示的是连字符，有时候表示的是加号。什么时候是连字符？什么时候是加呢？
**如果加号两边都是数值，此时是加。否则，就是连字符。** 

### 练习
```javascript
<script type="text/javascript">
    var a = "1";
    var b = 2;
    console.log(a + b);
</script>                                       //12
```
```javascript
var a = 1;		//数字
var b = 2;		//数字
console.log(a + b);     //3
```
```javascript
var a = 234234;
var b = 234323112;
console.log("a+b");			//由于加上了引号，所以就表示原样输出   a+b
```
```javascript
var a = 1;
var b = 2;
console.log("a" + b);    	//"a"就不是变量了！所以就是"a"+2 输出a2
```
```javascript
var a = 123;
var b = "123";
console.log(a + "b");             //123b
```

## 六、变量值的传递  
语句：
```javascript
a = b;
```
**将等号右边的值，赋给左边的变量；等号右边的变量，值不变。  
把b的值赋给a，b不变。**
### 练习  
```javascript
var a = 1;		    //定义a，并且赋值1
var b = 2;		    //定义b，并且赋值2
a = b;			    //就是将b的值给a，b的值不变。所以a就是2了，b是2不变。
console.log(a);	    //2
console.log(b);	    //2
```

```javascript
var a = 1;
var b = 2;
b = a;			//将a的值给b，a的值不变，所以b就是1了，a还是1不变
console.log(a); //1
console.log(b); //1
```

```javascript
var a = 1;			//定义a变量，值是1
var b = 2;			//定义b变量，值是2
a = b + 3;			//将5赋值给a，b的值还是2。
b = a + 4;			//将9赋值给b，a的值还是5
console.log(a);		//5
console.log(b);		//9
```

```javascript
					    //a		b       c
var a = 1;              //1
var b = 2;              //1     2
var c = 3;              //1     2       3
a = b + c;              //5     2       3

c = a * b;              //5     -2      -10
console.log(a);         //5
console.log(b);         //-2

```


## 七、运算符和表达式  
### 1、数学运算符
JavaScript中正确的表达式就是：
```javascript
(3 + 4 * 5) / (6 + 3)
```
+、*、/、( 都是运算符
这个式子叫做表达式。

   符号   |   意义    |
----------|----------|
    +     |    加    | 
    -     |    减    | 
    *     |    乘    | 
    /     |    除    | 
    %     |  取余数  | 
    ()    |   括号   |  
### 2、乘方和开根号  
```javascript
var a = Math.pow(3,4);
console.log(a);
```
Math是英语“数学”，pow就是“power”乘方的意思。
```javascript
var a = Math.sqrt(81);
console.log(a);
```
sqrt就是英语“开根号”的意思。

## 八、变量格式转换
### 1、用户的输入
```javascript
<script type="text/javascript">
    var a = prompt("哈哈请输入点什么东西吧");
    console.log(a);
</script>
```
prompt就是专门用来弹出能够让用户输入的对话框  
和alert很像，又不像：
```javascript
alert("从前有座山");                //直接使用，不需要变量
var a = prompt("请输入一个数字");   // 必须用一个变量，来接收用户输入的值
```
用户不管输入什么，都是字符串！
### 2、字符串→数字  
方法就是：  
```javascript
parseInt(“5”);
```
结果就是数字5。  
parse是英语里面的转换的意思，Int表示整数。注意拼写：
parseInt

**● parseInt带有自动净化的功能**：
```javascript
console.log(parseInt("365天每天都爱你10000次"));  //365
```
后面的中文自动消失，只保留最开头的数字。  

**● 自动带有截断小数功能（取整，不四舍五入）**
```javascript
console.log(parseInt(5.8));  //5
```

总结一下，parseInt是一个非常多功能的东西，可以将字符串转为数字，也可以将数字取整。
